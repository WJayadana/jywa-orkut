    JywaOrkut - v1.0.0

    jywa-orkut

    API wrapper untuk Order Kuota - Solusi mudah untuk integrasi layanan order kuota.

    📦 Installation

    npm install jywa-orkut

    🚀 Quick Start

    import { JywaOrkut } from 'jywa-orkut';

    const orkut = new JywaOrkut({
      apiKey: 'your-api-key',
      baseUrl: 'https://your-api-url.com'
    });

    // Example usage
    async function orderKuota() {
      try {
        const result = await orkut.orderKuota({
          phone: '08123456789',
          amount: 10000,
          provider: 'telkomsel'
        });
        console.log('Order successful:', result);
      } catch (error) {
        console.error('Order failed:', error);
      }
    }

    📋 Features

    • ✅ Easy-to-use API wrapper
    • ✅ TypeScript support
    • ✅ Error handling
    • ✅ QR Code generation
    • ✅ Multiple provider support
    • ✅ Comprehensive documentation

    📖 API Documentation

    Constructor

    const orkut = new JywaOrkut(config: JywaOrkutConfig);

    Methods

    • orderKuota(params) - Order kuota for specified phone number
    • checkStatus(orderId) - Check order status
    • getProviders() - Get available providers
    • generateQR(data) - Generate QR code

    🔧 Configuration
